# Sistema de Gestión de Empleados

## Descripción
Sistema simple para gestionar datos de empleados utilizando listas paralelas en Python.

## Características
- Gestión de empleados usando tres listas paralelas (nombres, edades, salarios)
- Validación de datos de entrada
- Interfaz de consola intuitiva
- Código estructurado siguiendo principios de Clean Code

## Funcionalidades
1. **Agregar empleado**: Valida que el empleado no exista previamente
2. **Buscar empleado**: Muestra información completa de un empleado específico
3. **Comparar salarios**: Encuentra y muestra el empleado con mayor salario
4. **Mostrar estadísticas**: Muestra datos agregados del sistema
5. **Listar empleados**: Muestra todos los empleados registrados
